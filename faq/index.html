よくある質問（FAQ）

お問い合わせの多い質問をまとめました。まずこちらで解決方法をお探しください。

🔒 データの安全性・保存について

Q 赤ちゃんの誕生日や離乳食データは外部に送信されますか？
いいえ、送信されません。0-TOOLsのすべての育児ツールは、処理がお使いの端末のブラウザ内だけで完結します。入力した赤ちゃんの誕生日・離乳食の記録・メモなどが外部サーバーやインターネット上に送信されることは一切ありません。
Q 入力したデータはどこに保存されますか？
お使いの端末のブラウザ（localStorage）に保存されます。アカウントやサーバーへの保存ではないため、ブラウザのデータを消去するとデータが失われます。大切なデータはエクスポート機能でバックアップしてください。
Q データが消えてしまいました
以下の操作を行うと、localStorageに保存されたデータが削除されます：
  • ブラウザの「履歴・Cookieの消去」「サイトデータの削除」
  • プライベートブラウズ（シークレットモード）の終了時
  • 端末の初期化・ブラウザのアンインストール
定期的にエクスポート機能でバックアップをお取りください。
Q データを別のスマホや端末に引き継げますか？
はい、エクスポート機能でデータをJSONファイルとして書き出し、新しい端末でインポートすることで引き継げます。機種変更の前にエクスポートしておくことをおすすめします。

📅 計算・機能について

Q 月齢・週数の計算方法は？
生後日数は誕生日を0日目として計算します。週数は7日ごと（生後7日で1週0日）、月齢は誕生日の同日から翌月の同日前日まで（例：1月15日生まれなら2月14日まで生後0ヶ月）として計算しています。
Q 離乳食の段階（初期・中期・後期）はどう判定されますか？
月齢をもとに自動判定しています。目安は以下の通りです（個人差があります）：
  • 初期：生後5〜6ヶ月頃
  • 中期：生後7〜8ヶ月頃
  • 後期：生後9〜11ヶ月頃
  • 完了期：生後12〜18ヶ月頃
実際の開始タイミングはお子様の発達状況や医師のアドバイスに合わせてご判断ください。

📋 基本的な使い方

Q スマホでも使えますか？
はい、iOS（Safari）・Android（Chrome）など主要なスマホブラウザに対応しています。スマホでの操作を最優先に設計されており、すべての機能をスマホから快適にご利用いただけます。
Q 無料で使えますか？登録は必要ですか？
完全無料・会員登録不要でお使いいただけます。すべての処理がブラウザ内で完結するためサーバー費用が発生せず、広告収入のみで無料提供を続けています。
Q 対応しているブラウザは？
以下の最新版ブラウザに対応しています：
  • Google Chrome（推奨）
  • Mozilla Firefox
  • Apple Safari（iOS含む）
  • Microsoft Edge
古いバージョンのブラウザでは正常に動作しない場合があります。

解決しない場合

上記のFAQで解決しない場合は、お問い合わせフォームよりご連絡ください。当サイトは個人運営のため、返信できない場合や大幅に遅れる場合があります。不具合の場合は、ブラウザ名・バージョン・端末の種類をあわせてご記入いただけると対応しやすくなります。
