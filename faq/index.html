よくある質問（FAQ）

お問い合わせの多い質問をまとめました。まずこちらで解決方法をお探しください。

🔒 セキュリティ・プライバシー

Q ファイルは外部に送信されますか？
いいえ、送信されません。0-TOOLs.JAPANのすべてのツールは、処理がお使いの端末のブラウザ内だけで完結します。PDF・画像などのファイルデータが外部サーバーやインターネット上に送信されることは一切ありません。機密性の高い書類も安心してご利用いただけます。
Q 個人情報は収集されますか？
ツールの利用中に処理されるファイルの内容は収集されません。ただし、Google AdSenseの広告配信のためにCookieが使用される場合があります。詳しくはプライバシーポリシーをご確認ください。
Q パスワード付きPDFは処理できますか？
パスワード（閲覧制限）が設定されているPDFは処理できません。あらかじめパスワードを解除してからご利用ください。

📋 基本的な使い方

Q スマホでも使えますか？
はい、iOS（Safari）・Android（Chrome）など主要なスマホブラウザに対応しています。スマホでの操作を最優先に設計されており、画面全体をタップしてファイルを選択できます。
Q 無料で使えますか？登録は必要ですか？
完全無料・会員登録不要でお使いいただけます。すべての処理がブラウザ内で完結するためサーバー費用が発生せず、無料で提供を続けることができています。
Q 複数のファイルをまとめて処理できますか？
はい、複数のPDFファイルをまとめて選択して処理することができます。ファイル選択時に複数のPDFを選ぶか、ドラッグ＆ドロップで複数ファイルを一度に投入してください。複数ファイルの場合、変換結果はZIPファイルでまとめてダウンロードできます。
Q 対応しているブラウザは？
以下の最新版ブラウザに対応しています：
  • Google Chrome（推奨）
  • Mozilla Firefox
  • Apple Safari（iOS含む）
  • Microsoft Edge
古いバージョンのブラウザでは正常に動作しない場合があります。

⚠️ トラブルシューティング

Q 処理が途中で止まってしまいます
ページ数が多いPDFや高解像度の画像が多く含まれるファイルを処理する際、端末のメモリが不足して処理が止まることがあります。以下をお試しください：
  • 品質設定を低めに変更する（PDF圧縮なら「スマホ」や「標準」）
  • ブラウザのタブを最小限に絞る
  • ページ数の多いPDFは分割して処理する
  • メモリの多い端末（PC等）で試してみる
Q 変換後のファイルが保存されません
スマホの場合、ブラウザの設定によってはダウンロードフォルダへの保存先が変わる場合があります。ダウンロードボタンをタップした後、端末の「ファイル」アプリや「写真」アプリ、または通知バーから保存先を確認してください。
Q PDFの文字が読めないほどぼやけてしまいます
PDF圧縮ツールでは、画質プリセットを上げることで文字のぼやけを改善できます。「標準」や「印刷」など、より高い設定をお試しください。なお、本ツールはPDFを画像として再変換するため、元のPDFよりも文字の鮮明さが低下する場合があります。文字の品質を重視する場合は「高画質（300dpi）」をご利用ください。
Q 圧縮してもファイルサイズが変わりません・大きくなりました
すでに高度に圧縮されたPDFや、もともとファイルサイズが小さいPDFの場合、圧縮後のサイズが元と同じかやや大きくなる場合があります。これはPDFを画像として再変換・再エンコードする仕組みによるものです。この場合、品質設定を「最小」に下げることで改善される場合があります。

解決しない場合

上記のFAQで解決しない場合は、お問い合わせフォームよりご連絡ください。当サイトは個人運営のため、返信できない場合や大幅に遅れる場合があります。不具合の場合は、ブラウザ名・バージョン・端末の種類をあわせてご記入いただけると対応しやすくなります。
