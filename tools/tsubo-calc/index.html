平米・坪・畳の換算について

平方メートル（㎡）は国際的な面積の単位です。日本の不動産では坪（つぼ）も広く使われており、1坪は約3.306㎡に相当します。マンションや戸建ての間取り図では㎡表記が一般的ですが、賃貸オフィスや土地の取引では坪が使われることが多くあります。

畳（じょう）は和室の広さを表す単位で、地域によって広さが異なります。関東で一般的な江戸間（1畳≒1.548㎡）と、関西で使われる京間（1畳≒1.82㎡）では同じ「6畳」でも広さが変わります。このツールでは江戸間・京間の両方を同時に表示します。

坪単価とは、1坪あたりの価格のことです。「月額賃料 ÷ 坪数」で計算でき、広さの異なる物件を比較する際の目安として使われます。

よくある質問

1坪 = 約3.306㎡（正確には3.30578㎡）です。逆に1㎡ = 約0.3025坪です。不動産の面積表記でよく使われる換算値で、「20坪 = 約66㎡」「30坪 = 約99㎡」が目安になります。
坪単価とは1坪（≒3.3㎡）あたりの価格のことです。賃料では「月額賃料 ÷ 坪数」で計算します。広さの異なる物件を比較するときの目安として使われます。たとえば月額15万円・50㎡（約15.1坪）の物件であれば、坪単価は約9,900円になります。
地域によって異なります。江戸間（関東）：約1.548㎡、中京間：約1.653㎡、京間（関西）：約1.82㎡です。同じ「6畳」でも、江戸間では約9.3㎡、京間では約10.9㎡と約1.6㎡の差があります。このツールでは江戸間と京間の両方を同時に表示します。
20坪 = 約66.1㎡（20 × 3.30578 = 66.12㎡）です。2LDK〜3LDKのマンションに相当する広さです。
30坪 = 約99.2㎡（30 × 3.30578 = 99.17㎡）です。戸建て住宅の延床面積として一般的な広さです。
100㎡ = 約30.25坪（100 ÷ 3.30578 = 30.25坪）です。広めのマンションや戸建て住宅に相当します。
