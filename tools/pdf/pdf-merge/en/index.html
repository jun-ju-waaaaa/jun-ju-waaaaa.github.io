    FAQ

    Is it free to use?

    Yes, completely free. No limits on file count or number of uses.

    Are my files uploaded anywhere?

    Your files are never sent to a server. All processing happens locally in your browser. We have no access to your documents.

    Does it work on iPhone and Android?

    Yes, it works in the browser on both iPhone and Android — no app needed. You can pick PDFs from the Files app (iOS) or Files app (Android) directly.

    How many files can I merge at once?

    There's no hard limit. Merging a very large number of files or large PDFs may take longer depending on your device's available memory.

    About this tool ▾

    What it does

    Combines multiple PDF files into a single PDF. Use the ↑↓ buttons to set the order before merging.

    How it works

    All processing is done in your browser using an open-source library (pdf-lib). No files leave your device.

    Why is it free?

    Because there are no server costs, we can offer this tool at no charge.

    Password-protected PDFs cannot be merged.
    Large files or many pages may take longer to process.
