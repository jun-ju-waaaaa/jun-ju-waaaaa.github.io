How to Measure

服（レディース・メンズ）

バスト・ウエスト・ヒップをメジャーで水平に測ります。レディース服は日本号数（5〜21）が基準で、海外ではUS・UK・EUサイズが混在します。メンズは身長（cm）をベースに変換するのが最も確実です。ブランドによって着丈・肩幅が異なるため、必ずサイズガイドの実寸も確認しましょう。

靴

かかとからつま先までの長さを紙の上で測ります（足長）。日本のcm表記が最も正確で、この表を使いUSサイズに変換できます。同じサイズでも幅（ワイズ）が異なるため、幅広の方はワイズ対応モデルを選ぶと失敗しにくいです。

指輪

細い紙を指に巻いて重なり部分に印を付け、長さを測ります。その長さ÷3.14で内径（mm）が分かります。指は朝と夜でむくみ具合が変わるため、測定は夕方がおすすめです。

ブラジャー

アンダーバスト（胸の下を一周した長さ）でバンドサイズが決まります。トップバストとの差でカップサイズが決まります（差5cm=A、10cm=B、15cm=C…）。海外ブランドではバンドサイズの表記が異なるため、この表で変換してから購入してください。

FAQ

海外通販で買う場合、どのサイズを選べばいいですか？ +
一般的に、US・EUサイズで迷ったら「普段の日本サイズをこの表で変換したサイズ」を基準に、レビューで「大きめ・小さめ」の口コミを確認して最終決定することをおすすめします。
ブランドによってサイズが違うのはなぜですか？ +
サイズ表記は国際標準がありますが、ブランドごとに独自の「ゆとり」設計があるため、同じ号数でも実寸が異なる場合があります。必ず各ブランドの公式サイズガイドも確認してください。
靴のサイズはcmとUSサイズどちらで選ぶべきですか？ +
日本人にはcm（実寸）での管理が最も正確です。海外ブランドを買う際はcmを基準にこのツールでUSサイズに変換し、ブランドの実寸表（インソール長）と照合するのが確実です。
指輪のサイズが分からない場合はどうすればいいですか？ +
細い紙を指に巻いて長さを測り、÷3.14で内径(mm)を計算できます。この表の「内径(mm)」列と照合してください。
