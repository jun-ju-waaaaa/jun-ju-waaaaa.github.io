JSON整形ツール

JSONをフォーマット（整形）・圧縮・バリデーション。エラーの行番号も表示。API開発やデバッグに役立つ無料ツールです。

よくある質問

JSON（JavaScript Object Notation）はデータを表現するテキスト形式です。WebAPIのレスポンスや設定ファイルなどで広く使われています。
整形（フォーマット）はインデント・改行を追加して読みやすくすること。圧縮（minify）は余分な空白・改行を削除してデータサイズを小さくすること。APIの通信には圧縮形式が使われます。
エラーメッセージに行番号が表示されます。よくある問題：①文字列がダブルクォート（"）で囲まれていない、②末尾にカンマが残っている、③括弧の対応が合っていない。
いいえ。処理はすべてブラウザ内で行われます。機密情報が含まれるJSONも安心して使えます。
