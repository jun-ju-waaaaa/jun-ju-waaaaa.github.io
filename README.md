## アーカイブ（noindex済み・非公開）

以下のツールはサイト方針変更（育児特化）に伴いnoindex済みです。HTMLは保持しています。

- 年齢計算・BMI・消費税・パーセント・時間・営業日・平米坪・単価・サイズ変換
- 文字数カウント・Base64・JSON整形
- 画像変換・ファビコン・QRコード・カラーコード変換
- PDF圧縮・結合・回転・JPEG変換
- 固定資産税・都市計画税計算（→ [fudosan-tools.com](https://fudosan-tools.com) に移転）
